# Superstore Sales Dashboard

## Project Overview

This Power BI project analyzes retail sales performance using the Superstore dataset.

## Objectives

- Analyze sales and profit trends
- Identify top-performing products
- Compare regional performance
- Study customer segments
- Evaluate discount impact on profitability

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Average Order Value

## Dashboard Features

- Sales Trend Analysis
- Region Performance
- Category Performance
- Top 10 Products
- Customer Segment Analysis
- Interactive Filters

## Business Insights

- Technology category generated highest sales.
- Consumer segment contributed the largest revenue share.
- High discounts negatively impacted profitability.
